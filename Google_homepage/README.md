# Google Homepage Clone

A simple recreation of the Google homepage using HTML and CSS.

## Overview

This project replicates the classic Google search homepage layout, featuring:

- Google logo
- Search input field
- "Google Search" and "I'm Feeling Lucky" buttons

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling for the page
- `google.png` - Google logo image

## Usage

Open `index.html` in any web browser to view the page.

## Technologies

- HTML5
- CSS3
